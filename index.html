Responsive Design

This project was a study into the world of responsive design. Accessibility is key to gain the most exposure to all of the devices available in the world that can utilize the internet. The main goal in this project was to create a responsive desing that could be viewed from different viewport sizes built upon a previous project that we had been working on in the first week. Utilizing my new skills and understandings of the power of media queries as well as parent:child relationship and fluid layout, I was able to acheive MVP for the assignment and gain the experience necessary to continue forward.
